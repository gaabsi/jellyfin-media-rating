Media Rating

Plugin de notation style Netflix — dislike / like / love.

Les votes sont stockés de manière sécurisée dans la configuration du plugin.

Intégration Seerr

Connecte ton instance Seerr / Overseerr pour activer la recherche dans le tab "Mes votes".

Settings → General → API Key dans Seerr.

Intégration TMDB

Clé API TMDB (v3) — utilisée pour récupérer les logos des films/séries sur la page détail Seerr. Génération gratuite sur themoviedb.org/settings/api.

Champ "API Key (v3 auth)" sur la page TMDB.

Informations

L'onglet "Mes votes" est automatiquement ajouté à votre page d'accueil Jellyfin.

Le widget de vote remplace le bouton "Bande-annonce" sur la page de détails des médias.
