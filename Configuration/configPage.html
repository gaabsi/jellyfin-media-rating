Media Rating

Plugin de notation style Netflix — dislike / like / love.

Les votes sont stockés de manière sécurisée dans la configuration du plugin.

Informations

L'onglet "Mes votes" est automatiquement ajouté à votre page d'accueil Jellyfin.

Le widget de vote remplace le bouton "Bande-annonce" sur la page de détails des médias.
